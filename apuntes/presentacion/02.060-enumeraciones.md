# Enumeraciones en C#

Reemplaza números y strings mágicos por valores explícitos

---

## Introducción

```csharp
// Sin enum — ¿qué significa 2 o "A"?
if (orden.Estado == 2) { ... }
if (usuario.Rol == "A") { ... }

// Con enum — claro y explícito
if (orden.Estado == EstadoOrden.Enviado) { ... }
if (usuario.Rol == RolUsuario.Administrador) { ... }
```

Los enums en C# son **tipos por valor** basados en un entero subyacente (por defecto `int`).

---

## Declaración básica

```csharp
public enum DiaSemana {
    Lunes,      // 0
    Martes,     // 1
    Miercoles,  // 2
    Jueves,     // 3
    Viernes,    // 4
    Sabado,     // 5
    Domingo     // 6
}

public enum EstadoOrden {
    Pendiente = 0,
    Confirmado = 1,
    Enviado = 2,
    Entregado = 3,
    Cancelado = 99
}
```

---

## Tipos subyacentes

```csharp
public enum CodigoError : short {
    Ninguno = 0,
    Generico = 1000,
    NoEncontrado = 1001,
    SinPermiso = 1002
}
```

| Tipo                | Cuándo usarlo                   |
|--------------------|---------------------------------|
| `int` (default)    | Caso general                    |
| `byte`             | Pocos valores, serialización    |
| `short`            | Valores medianos                |
| `long`             | Flags con muchos bits           |

## Uso básico

```csharp
DiaSemana hoy = DiaSemana.Miercoles;
var mañana = DiaSemana.Jueves;

// Comparación
if (hoy == DiaSemana.Sabado || hoy == DiaSemana.Domingo)
    Console.WriteLine("Es fin de semana");

// Switch expression
string tipo = hoy switch {
    DiaSemana.Sabado or DiaSemana.Domingo => "Fin de semana",
    DiaSemana.Lunes => "Inicio de semana",
    _ => "Día hábil"
};
```

## Conversión enum ↔ int ↔ string

```csharp
// Enum → int (cast explícito)
int valor = (int)EstadoOrden.Enviado;  // 2

// int → enum (cast explícito)
EstadoOrden estado = (EstadoOrden)2;  // EstadoOrden.Enviado

// Enum → string
string nombre = DiaSemana.Lunes.ToString();  // "Lunes"

// string → enum (Parse — lanza excepción)
DiaSemana dia = Enum.Parse<DiaSemana>("Martes");

// string → enum (TryParse — seguro)
if (Enum.TryParse<DiaSemana>("Viernes", out DiaSemana resultado))
    Console.WriteLine($"Parseado: {resultado}");

// Ignorar mayúsculas
Enum.TryParse<DiaSemana>("lunes", ignoreCase: true, out DiaSemana d);
```

## Verificar si un valor es válido

```csharp
// Enum.IsDefined — verifica que corresponda a un miembro definido
bool valido1 = Enum.IsDefined<EstadoOrden>(EstadoOrden.Enviado);  // true
bool valido2 = Enum.IsDefined<EstadoOrden>((EstadoOrden)50);      // false

// Validar datos externos (JSON, BD, API)
static EstadoOrden ValidarEstado(int valorExterno) {
    if (!Enum.IsDefined<EstadoOrden>((EstadoOrden)valorExterno))
        throw new ArgumentOutOfRangeException(nameof(valorExterno));
    return (EstadoOrden)valorExterno;
}
```

## Reflexión sobre enums

```csharp
// Obtener todos los valores
DiaSemana[] todos = Enum.GetValues<DiaSemana>();

// Obtener todos los nombres
string[] nombres = Enum.GetNames<DiaSemana>();

// Iterar todos los valores
foreach (DiaSemana dia in Enum.GetValues<DiaSemana>())
    Console.WriteLine($"{(int)dia}: {dia}");

// Con LINQ
var diasLaborales = Enum.GetValues<DiaSemana>()
    .Where(d => d != DiaSemana.Sabado && d != DiaSemana.Domingo)
    .ToList();
```

## Atributos en enums — `[Description]`

```csharp
using System.ComponentModel;

public enum EstadoPedido {
    [Description("Pendiente de pago")]
    PendientePago = 0,
    
    [Description("Pago confirmado")]
    PagoConfirmado = 1,
    
    [Description("En preparación")]
    EnPreparacion = 2
}

// Método de extensión para obtener la descripción
public static string Descripcion(this Enum valor) {
    var campo = valor.GetType().GetField(valor.ToString());
    var attr = campo?.GetCustomAttributes(typeof(DescriptionAttribute), false)
                     .OfType<DescriptionAttribute>()
                     .FirstOrDefault();
    return attr?.Description ?? valor.ToString();
}

// Uso
Console.WriteLine(EstadoPedido.EnPreparacion.Descripcion());  // "En preparación"
```

## Flags — enums de bit a bit

Cuando un valor es **combinación** de varios miembros:

```csharp
[Flags]
public enum Permiso {
    Ninguno = 0,
    Leer = 1,      // 0001
    Escribir = 2,  // 0010
    Ejecutar = 4,  // 0100
    Eliminar = 8,  // 1000
    
    Completo = Leer | Escribir | Ejecutar | Eliminar
}

// Crear combinaciones
Permiso admin = Permiso.Completo;
Permiso editor = Permiso.Leer | Permiso.Escribir;

// Verificar con HasFlag
bool puedeEscribir = editor.HasFlag(Permiso.Escribir);  // true
bool puedeEliminar = editor.HasFlag(Permiso.Eliminar);  // false

// Agregar y quitar permisos
editor |= Permiso.Ejecutar;    // agregar
editor &= ~Permiso.Escribir;   // quitar
```

## Pattern matching con enums

```csharp
public enum Prioridad { Baja, Media, Alta, Critica }

record Incidente(string Titulo, Prioridad Prioridad, bool EstaResuelto);

static TimeSpan TiempoRespuesta(Incidente inc) => inc switch {
    { EstaResuelto: true } => TimeSpan.Zero,
    { Prioridad: Prioridad.Critica } => TimeSpan.FromMinutes(15),
    { Prioridad: Prioridad.Alta } => TimeSpan.FromHours(1),
    { Prioridad: Prioridad.Media } => TimeSpan.FromHours(4),
    { Prioridad: Prioridad.Baja } => TimeSpan.FromDays(1),
    _ => throw new ArgumentOutOfRangeException()
};
```

## Serialización de enums

**System.Text.Json:**
```csharp
using System.Text.Json.Serialization;

// Por defecto — serializa como número
var json1 = JsonSerializer.Serialize(p);  // {"Documento":1}

// Como string — más legible
var opciones = new JsonSerializerOptions {
    Converters = { new JsonStringEnumConverter() }
};
var json2 = JsonSerializer.Serialize(p, opciones);  // {"Documento":"Pasaporte"}

// Nombres personalizados
public enum Moneda {
    [JsonStringEnumMemberName("ARS")] PesoArgentino,
    [JsonStringEnumMemberName("USD")] DolarEstadounidense
}
```

## Buenas prácticas

1. **Definir un valor 0 significativo**
```csharp
// ✓ BIEN
public enum Color { Ninguno = 0, Rojo = 1, Verde = 2, Azul = 3 }
```

2. **Siempre validar valores externos**
```csharp
if (!Enum.IsDefined<T>(valor))
    throw new ArgumentException("Valor inválido");
```

3. **Preferir switch exhaustivo** (sin default cuando sea posible)

4. **Documentar los valores** con comentarios XML
