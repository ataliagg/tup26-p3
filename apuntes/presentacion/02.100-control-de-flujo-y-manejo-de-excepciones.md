# Control de Flujo y Manejo de Excepciones

Ejecutando instrucciones según condiciones

---

## `if` / `else if` / `else`

```csharp
if (temperatura > 35) {
    Console.WriteLine("Calor extremo");
}
else if (temperatura > 25) {
    Console.WriteLine("Calor agradable");
}
else if (temperatura > 15) {
    Console.WriteLine("Temperatura templada");
}
else {
    Console.WriteLine("Frío");
}
```

**Condiciones compuestas:**
```csharp
if (edad >= 18 && tieneDni)
    Console.WriteLine("Puede votar");

if (edad < 12 || edad > 65)
    Console.WriteLine("Tarifa reducida");

if (!tieneDni)
    Console.WriteLine("Documentación incompleta");
```

---

## Operador ternario `? :`

Expresión condicional compacta:

```csharp
// if/else clásico
string categoria;
if (edad >= 18)
    categoria = "adulto";
else
    categoria = "menor";

// Ternario
string categoria = edad >= 18 ? "adulto" : "menor";

// Como retorno
static string Clasificar(int n) => n > 0 ? "positivo" : n < 0 ? "negativo" : "cero";
```

## `switch` instrucción

```csharp
switch (diaSemana) {
    case 1:
        Console.WriteLine("Lunes");
        break;
    case 2:
        Console.WriteLine("Martes");
        break;
    case 6:
    case 7:  // fall-through
        Console.WriteLine("Fin de semana");
        break;
    default:
        Console.WriteLine("Día inválido");
        break;
}
```

---

## `switch` expression — forma moderna

```csharp
string nombre = diaSemana switch {
    1       => "Lunes",
    2       => "Martes",
    3       => "Miércoles",
    6 or 7  => "Fin de semana",
    _       => "Día inválido"
};

// Con pattern matching
string tipo = figura switch {
    Circulo c => $"Círculo radio {c.Radio}",
    Rectangulo r => $"Rectángulo {r.Ancho}×{r.Alto}",
    _ => "Desconocida"
};
```

## Pattern matching en `if`

```csharp
object valor = 42;

if (valor is int numero)
    Console.WriteLine($"Es entero: {numero * 2}");

if (valor is int n and > 0)
    Console.WriteLine($"Entero positivo: {n}");

string? nombre = ObtenerNombre();
if (nombre is not null)
    Console.WriteLine(nombre.ToUpper());
```

## `while` — mientras se cumpla

```csharp
int intentos = 0;
while (intentos < 3) {
    Console.Write("Contraseña: ");
    string? input = Console.ReadLine();
    if (input == "1234") {
        Console.WriteLine("Acceso concedido");
        break;
    }
    intentos++;
}
```

---

## `do while` — al menos una vez

```csharp
int numero;
do {
    Console.Write("Ingresá número 1-10: ");
    string? entrada = Console.ReadLine();
    int.TryParse(entrada, out numero);
}
while (numero < 1 || numero > 10);

Console.WriteLine($"Ingresaste: {numero}");
```

## `for` — con contador

```csharp
// Forma estándar
for (int i = 0; i < 10; i++)
    Console.Write($"{i} ");

// Hacia atrás
for (int i = 9; i >= 0; i--)
    Console.Write($"{i} ");

// Paso diferente
for (int i = 0; i <= 100; i += 10)
    Console.Write($"{i} ");

// Con array
string[] nombres = ["Ana", "Carlos", "Laura"];
for (int i = 0; i < nombres.Length; i++)
    Console.WriteLine($"[{i}] {nombres[i]}");
```

---

## `foreach` — recorrer colecciones

```csharp
var frutas = new List<string> { "manzana", "banana", "cereza" };

foreach (var fruta in frutas)
    Console.WriteLine(fruta);

// Con índice
foreach (var (fruta, i) in frutas.Select((f, i) => (f, i)))
    Console.WriteLine($"[{i}] {fruta}");

// Diccionario
var edades = new Dictionary<string, int> {
    ["Ana"] = 25, ["Carlos"] = 30
};

foreach (var (nombre, edad) in edades)
    Console.WriteLine($"{nombre}: {edad}");
```

## Sentencias de salto

**`break`** — salir del bucle:
```csharp
int[] numeros = [3, 7, 12, 5, 9];
for (int i = 0; i < numeros.Length; i++) {
    if (numeros[i] == 12) {
        Console.WriteLine($"Encontrado en {i}");
        break;
    }
}
```

**`continue`** — saltar a la siguiente iteración:
```csharp
for (int i = 0; i <= 10; i++) {
    if (i % 2 != 0)
        continue;  // salta impares
    Console.Write($"{i} ");
}
// 0 2 4 6 8 10
```

**`return`** — salir del método:
```csharp
static double Dividir(double a, double b) {
    if (b == 0)
        return double.NaN;
    return a / b;
}
```

## Try / Catch / Finally

```csharp
try {
    string contenido = File.ReadAllText("datos.txt");
    Console.WriteLine(contenido);
}
catch (FileNotFoundException ex) {
    Console.WriteLine($"Archivo no encontrado: {ex.FileName}");
}
catch (UnauthorizedAccessException ex) {
    Console.WriteLine($"Sin permisos: {ex.Message}");
}
catch (Exception ex) {
    Console.WriteLine($"Error: {ex.Message}");
}
finally {
    Console.WriteLine("Operación finalizada.");
}
```

Los `catch` se evalúan **de arriba a abajo** — excepciones más específicas primero.

---

## `finally` — Siempre se ejecuta

```csharp
using var lector = new StreamReader("datos.txt");
try {
    string contenido = lector.ReadToEnd();
    Console.WriteLine(contenido);
}
catch (FileNotFoundException) {
    Console.WriteLine("Archivo no existe.");
}
// Dispose() se llama automáticamente al salir del bloque using
```

## Lanzar excepciones — `throw`

```csharp
static double Raiz(double n) {
    if (n < 0)
        throw new ArgumentOutOfRangeException(nameof(n), 
            "No se puede calcular raíz de negativo");
    return Math.Sqrt(n);
}

// Relanzar preservando el stack trace
try {
    Raiz(-5);
}
catch (ArgumentOutOfRangeException ex) {
    Console.WriteLine($"Error: {ex.Message}");
    throw;  // relanza la excepción original
}
```

---

## `throw` como expresión

```csharp
// En ternario
static string Validar(string? nombre)
    => nombre is null ? throw new ArgumentNullException(nameof(nombre)) : nombre;

// Con operador ??
string nombre = ObtenerNombre() 
    ?? throw new InvalidOperationException("Nombre requerido");

// En switch expression
static double Dividir(double a, double b) => b == 0
    ? throw new DivideByZeroException()
    : a / b;
```

## Excepciones personalizadas

```csharp
public class SaldoInsuficienteException : Exception {
    public decimal SaldoActual { get; }
    public decimal MontoSolicitado { get; }
    
    public SaldoInsuficienteException(decimal saldo, decimal monto)
        : base($"Saldo: {saldo}, Solicitado: {monto}") {
        SaldoActual = saldo;
        MontoSolicitado = monto;
    }
}

// Uso
try {
    var cuenta = new CuentaBancaria(500);
    cuenta.Extraer(800);
}
catch (SaldoInsuficienteException ex) {
    Console.WriteLine($"No hay dinero. Disponible: {ex.SaldoActual}");
}
```

## `when` en catch — filtro de excepción

```csharp
try {
    await LlamarApiAsync();
}
catch (HttpRequestException ex) when (ex.StatusCode == 404) {
    Console.WriteLine("Recurso no encontrado");
}
catch (HttpRequestException ex) when (ex.StatusCode == 401) {
    Console.WriteLine("Sin autorización");
}
catch (HttpRequestException ex) {
    Console.WriteLine($"Error HTTP: {ex.StatusCode}");
}
```

## Buenas prácticas

**Capturar excepciones específicas:**
```csharp
// ✓ BIEN
catch (FileNotFoundException ex) { }

// ✗ EVITAR — capturar Exception genérica
catch (Exception) { }
```

**Validar parámetros:**
```csharp
static void Procesar(string nombre) {
    ArgumentNullException.ThrowIfNull(nombre);
    ArgumentException.ThrowIfNullOrWhiteSpace(nombre);
    // ...
}
```

**Usar TryParse para conversiones esperadas:**
```csharp
if (int.TryParse(entrada, out int numero))
    Console.WriteLine($"Número: {numero}");
else
    Console.WriteLine("Entrada inválida");
```

## Guard clauses — retornos tempranos

```csharp
// Sin guard clauses — anidado
static void Procesar(string? nombre, int edad) {
    if (nombre != null) {
        if (edad > 0) {
            // lógica principal
        }
    }
}

// Con guard clauses — más limpio
static void Procesar(string? nombre, int edad) {
    if (nombre is null) return;
    if (edad <= 0) return;
    // lógica principal
}
```
