# Switch Expression y Pattern Matching

Ramificación expresiva y sin anidamiento

---

## Introducción

El **pattern matching** examina un valor según su **estructura**, **tipo** o **contenido**.

```csharp
// Antes — muchas variables temporales
string resultado;
if (figura is Circulo c)
    resultado = $"Círculo radio {c.Radio}";
else if (figura is Rectangulo r)
    resultado = $"Rectángulo {r.Ancho}×{r.Alto}";
else
    resultado = "Desconocida";

// Ahora — switch expression
string resultado = figura switch {
    Circulo c => $"Círculo radio {c.Radio}",
    Rectangulo r => $"Rectángulo {r.Ancho}×{r.Alto}",
    _ => "Desconocida"
};
```

---

## Anatomía del switch expression

```csharp
valor switch {
    patron1             => resultado1,
    patron2 when guarda => resultado2,
    _                   => porDefecto
}
```

| Aspecto        | `switch` instrucción | `switch` expression |
|---|---|---|
| Produce        | Ejecuta sentencias   | Un valor            |
| `break`        | Obligatorio          | No existe           |
| `default`      | Keyword              | `_` (wildcard)      |
| Exhaustividad  | No verificada        | Verificada          |

## Constant pattern — literales

```csharp
string comando = "salir";

string respuesta = comando switch {
    "ayuda" or "help" or "?" => MostrarAyuda(),
    "salir" or "exit" => "Hasta luego.",
    "limpiar" or "cls" => Limpiar(),
    "" => "Comando vacío.",
    _ => $"Desconocido: '{comando}'"
};
```

Con números:
```csharp
int codigo = 404;
string mensaje = codigo switch {
    200 => "OK",
    201 => "Creado",
    404 => "No encontrado",
    500 => "Error interno",
    _ => $"Código: {codigo}"
};
```

## Type pattern — por tipo

```csharp
object valor = 3.14;

string descripcion = valor switch {
    int n => $"entero: {n}",
    double d => $"decimal: {d:F2}",
    string s => $"cadena de {s.Length} caracteres",
    bool b => $"booleano: {(b ? "verdadero" : "falso")}",
    null => "nulo",
    _ => $"tipo: {valor.GetType().Name}"
};
```

## Property pattern — propiedades

```csharp
record Pedido(decimal Total, string Estado, bool EsUrgente);

static decimal DescuentoPedido(Pedido p) => p switch {
    { Estado: "cancelado" } => 0,
    { Total: > 10_000, EsUrgente: false } => p.Total * 0.15m,
    { Total: > 5_000 } => p.Total * 0.10m,
    { EsUrgente: true } => p.Total * 0.05m,
    _ => 0
};

// Con anidamiento
record Cliente(string Nombre, Direccion? Direccion);
record Direccion(string Ciudad, string Pais);

static decimal Descuento(Cliente c) => c switch {
    { Direccion.Pais: "AR", Nombre: "VIP" } => 0.20m,
    { Direccion.Ciudad: "Tucumán" } => 0.05m,
    _ => 0
};
```

## Relational pattern — comparaciones

```csharp
static string ClasificarEdad(int edad) => edad switch {
    < 0 => throw new ArgumentException(),
    < 3 => "Bebé",
    < 13 => "Niño",
    < 18 => "Adolescente",
    < 30 => "Joven adulto",
    < 60 => "Adulto",
    _ => "Adulto mayor"
};

static string ClasificarTemperatura(double temp) => temp switch {
    < 0 => "Bajo cero",
    < 15 => "Frío",
    < 25 => "Templado",
    < 35 => "Caluroso",
    _ => "Muy caluroso"
};
```

## Logical pattern — `and`, `or`, `not`

```csharp
// and — ambos patrones deben coincidir
static bool EsAdultoJoven(int edad) => edad is >= 18 and <= 35;

// or — al menos uno debe coincidir
static bool EsFinDeSemana(DayOfWeek dia)
    => dia is DayOfWeek.Saturday or DayOfWeek.Sunday;

// not — el patrón NO debe coincidir
static bool EsLaboral(DayOfWeek dia)
    => dia is not (DayOfWeek.Saturday or DayOfWeek.Sunday);

// En switch expression
static string ZonaHoraria(int hora) => hora switch {
    >= 0 and < 6 => "Madrugada",
    >= 6 and < 12 => "Mañana",
    >= 12 and < 14 => "Mediodía",
    >= 14 and < 18 => "Tarde",
    >= 18 and < 21 => "Noche",
    _ => throw new ArgumentException()
};
```

## Tuple pattern — múltiples valores

```csharp
// Dos variables simultáneamente
static string Describir(bool esDia, bool llueve)
    => (esDia, llueve) switch {
        (true, false) => "Día soleado",
        (true, true) => "Día lluvioso",
        (false, false) => "Noche estrellada",
        (false, true) => "Noche tormentosa"
    };

// Tabla de verdad semántica
record Compra(decimal Total, string Categoria, string Cliente);

static decimal Descuento(Compra c) => (c.Cliente, c.Categoria) switch {
    ("VIP", "electrónica") => c.Total * 0.15m,
    ("VIP", _) => c.Total * 0.10m,
    (_, "electrónica") => c.Total * 0.05m,
    (_, _) => 0
};
```

## Positional pattern — deconstrucción

```csharp
record Punto(double X, double Y);

static string PosicionPunto(Punto p) => p switch {
    (0, 0) => "Origen",
    (var x, 0) => $"Eje X en {x}",
    (0, var y) => $"Eje Y en {y}",
    (> 0, > 0) => "Cuadrante I",
    (< 0, > 0) => "Cuadrante II",
    (< 0, < 0) => "Cuadrante III",
    (> 0, < 0) => "Cuadrante IV",
    _ => "Indefinido"
};
```

## List pattern — colecciones (C# 11+)

```csharp
// Sintaxis: [patron1, patron2, ..., patronN]
// .. = "cualquier cantidad de elementos"

static string AnalizarLista(int[] nums) => nums switch {
    [] => "vacía",
    [var x] => $"un elemento: {x}",
    [var x, var y] => $"dos elementos: {x} y {y}",
    [1, 2, 3] => "exactamente [1, 2, 3]",
    [> 0, > 0, > 0] => "tres positivos",
    [var primero, ..] => $"empieza con {primero}",
    [var primero, .., var ultimo] => $"del {primero} al {ultimo}",
    _ => "otra cosa"
};

// Con strings
static string ClasificarArgs(string[] args) => args switch {
    [] => "Sin argumentos",
    ["--help"] or ["-h"] => "Ayuda",
    ["--version"] => "Versión",
    [var cmd, .. var resto] => $"Comando: {cmd} con {resto.Length} args"
};
```

## `var` pattern — capturar siempre

```csharp
record Empleado(string Nombre, decimal Salario, int Años);

static string CategoriaEmpleado(Empleado e) => e switch {
    // Capturar propiedades para usar en guardia
    { Años: var a, Salario: var s } when a > 10 && s > 150_000 
        => "Ejecutivo senior",
    
    { Años: var a } when a > 10 
        => "Senior",
    
    { Salario: var s } when s > 100_000 
        => "Alto valor",
    
    { Nombre: var n } when n.StartsWith("Dr.") 
        => "Especialista",
    
    _ => "Empleado estándar"
};
```

## Guardias `when` — condición adicional

```csharp
record Transaccion(decimal Monto, string Moneda, DateTime Fecha);

static string Clasificar(Transaccion t) => t switch {
    { Moneda: "USD", Monto: var m } when m > 10_000
        => "Transferencia USD de alto valor",
    
    { Moneda: "ARS", Monto: var m } when m > 1_000_000
        => "Gran operación en pesos",
    
    _ when t.Fecha.DayOfWeek is DayOfWeek.Saturday or DayOfWeek.Sunday
        => "Transacción en fin de semana",
    
    _ => "Transacción estándar"
};
```

> La guardia solo se evalúa si el patrón coincide.

## Exhaustividad — cobertura de casos

El compilador verifica que cubras todos los casos posibles:

```csharp
// Error — falta cubrir false
static string Describir(bool b) => b switch {
    true => "verdadero"
    // Error CS8509: switch expression does not handle all possible values
};

// Correcto — ambos valores cubiertos
static string Describir(bool b) => b switch {
    true => "verdadero",
    false => "falso"
};
```

## Patrones con `is`

Todos los patrones funcionan también con `is`:

```csharp
object obj = new List<int> { 1, 2, 3 };

if (obj is List<int> lista && lista.Count > 0)
    Console.WriteLine($"Lista con {lista.Count} elementos");

record Persona(string Nombre, int Edad);
var persona = new Persona("Ana", 25);

if (persona is { Nombre: "Ana", Edad: >= 18 })
    Console.WriteLine("Ana es adulta");

if (persona is not { Edad: < 18 })
    Console.WriteLine("No es menor");
```

## Ejemplo integrador — Procesador de comandos

```csharp
record Comando(string Nombre, string[] Args);

static string EjecutarComando(Comando cmd) => cmd switch {
    { Nombre: "help" or "h", Args: [] }
        => MostrarAyuda(),
    
    { Nombre: "get", Args: [var recurso] }
        => ObtenerRecurso(recurso),
    
    { Nombre: "get", Args: [var recurso, "--format", var fmt] }
        => ObtenerFormateado(recurso, fmt),
    
    { Nombre: "delete", Args: [var id] } when int.TryParse(id, out _)
        => EliminarRecurso(id),
    
    { Nombre: "list", Args: ["--filter", var filtro, ..] }
        => ListarFiltrado(filtro),
    
    { Nombre: var cmd, Args: var args } when args.Length > 10
        => $"Demasiados argumentos para '{cmd}'",
    
    { Nombre: var desconocido }
        => $"Comando desconocido: '{desconocido}'"
};
```

## Tabla de referencia de patrones

| Patrón      | Sintaxis                              | Coincide cuando           |
|-------------|---------------------------------------|---------------------------|
| Constante   | `42`, `"texto"`, `true`, `null`       | Igual a la constante      |
| Tipo        | `string s`, `int n`                   | Es del tipo especificado  |
| Propiedad   | `{ Nombre: "Ana", Edad: > 18 }`       | Propiedades coinciden     |
| Relacional  | `> 5`, `<= 10`, `>= 0`                | Comparación verdadera     |
| Lógico      | `and`, `or`, `not`                    | Combinación de patrones   |
| Tuple       | `(> 0, < 0)`, `(true, false)`         | Cada elemento coincide    |
| Posicional  | `(var x, var y)`                      | Deconstrucción exitosa    |
| Lista       | `[var first, .., var last]`           | Estructura coincide       |
| `var`       | `var x`                               | Siempre, captura valor    |
| Wildcard    | `_`                                   | Siempre, no captura       |
| Guardia     | `pattern when condicion`              | Patrón + condición true   |
