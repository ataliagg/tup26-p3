# Clases y Objetos en C#

Fundamento de la programación orientada a objetos

---

## Declaración básica de una clase

```csharp
public class Persona {
    // Campos (datos)
    public string Nombre;
    public int Edad;
    
    // Método (comportamiento)
    public void Saludar() {
        Console.WriteLine($"Hola, soy {Nombre}");
    }
}

// Crear una instancia (objeto)
var persona = new Persona {
    Nombre = "Ana",
    Edad = 25
};
persona.Saludar();  // "Hola, soy Ana"
```

---

## Propiedades (Properties)

La forma moderna de acceder a datos con control.

```csharp
public class Persona {
    // Campo privado
    private int edad;
    
    // Propiedad con getter y setter
    public int Edad {
        get { return edad; }
        set { edad = value >= 0 && value <= 150 ? value : edad; }
    }
    
    // Auto-property — más compacta
    public string Nombre { get; set; }
    
    // Solo lectura
    public string Id { get; private set; } = Guid.NewGuid().ToString();
    
    // Con lógica
    public string ResumenVida => $"{Nombre} ({Edad} años)";
}
```

## Constructores

Métodos especiales que inicializan un objeto.

```csharp
public class Persona {
    public string Nombre { get; set; }
    public int Edad { get; set; }
    // Constructor sin parámetros
    public Persona() {
        Edad = 0;
    }
    // Constructor con parámetros
    public Persona(string nombre, int edad) {
        Nombre = nombre;
        Edad = edad;
    }
    // Constructor que llama a otro
    public Persona(string nombre) : this(nombre, 0) { }
}
// Uso
var p1 = new Persona();
var p2 = new Persona("Ana", 25);
var p3 = new Persona("Carlos");
```

## Herencia

Una clase puede **heredar** de otra.

```csharp
public class Empleado : Persona {
    public string Empresa { get; set; }
    public decimal Salario { get; set; }
    
    public void TrabajarEn(string proyecto) {
        Console.WriteLine($"{Nombre} trabaja en {proyecto}");
    }
}

var emp = new Empleado {
    Nombre = "Ana",
    Edad = 30,
    Empresa = "TechCorp",
    Salario = 50_000
};
```

## Métodos virtuales y override

Permite que la clase derivada **redefina** el comportamiento.

```csharp
public class Persona {
    public virtual string Describirse() {
        return $"{Nombre}, {Edad} años";
    }
}

public class Empleado : Persona {
    public string Empresa { get; set; }
    
    public override string Describirse() {
        return base.Describirse() + $", trabaja en {Empresa}";
    }
}

Persona p = new Persona { Nombre = "Ana", Edad = 25 };
Console.WriteLine(p.Describirse());  // "Ana, 25 años"

Empleado e = new Empleado { 
    Nombre = "Carlos", 
    Edad = 30,
    Empresa = "TechCorp"
};
Console.WriteLine(e.Describirse());  // "Carlos, 30 años, trabaja en TechCorp"
```

## Clases abstractas

Una clase que **no puede instanciarse** pero define estructura.

```csharp
public abstract class Figura {
    public abstract double CalcularArea();
}

public class Circulo : Figura {
    public double Radio { get; set; }
    
    public override double CalcularArea() {
        return Math.PI * Radio * Radio;
    }
}

// var fig = new Figura();  // ✗ Error — es abstracta
var circ = new Circulo { Radio = 5 };
Console.WriteLine(circ.CalcularArea());
```

## Static — datos y métodos compartidos

Pertenecen a la **clase**, no a las instancias.

```csharp
public class Contador {
    private static int instancias = 0;
    
    public Contador() {
        instancias++;
    }
    
    public static int CantidadInstancias {
        get { return instancias; }
    }
}

var c1 = new Contador();
var c2 = new Contador();
var c3 = new Contador();

Console.WriteLine(Contador.CantidadInstancias);  // 3
```

## Métodos estáticos

No necesitan una instancia.

```csharp
public class Matematica {
    public static int Maximo(int a, int b) {
        return a > b ? a : b;
    }
    
    public static double Distancia(double x1, double y1, double x2, double y2) {
        return Math.Sqrt(Math.Pow(x2 - x1, 2) + Math.Pow(y2 - y1, 2));
    }
}

Console.WriteLine(Matematica.Maximo(5, 10));        // 10
Console.WriteLine(Matematica.Distancia(0, 0, 3, 4)); // 5
```

## Encapsulación (access modifiers)

| Modificador | Visibilidad                           |
|------------|---------------------------------------|
| `public`   | Visible en todas partes               |
| `private`  | Visible solo en la clase             |
| `protected` | Visible en la clase y clases derivadas |
| `internal` | Visible solo dentro del assembly     |

```csharp
public class Cuenta {
    private decimal saldo;  // privado
    
    public decimal ObtenerSaldo() {
        return saldo;
    }
    
    public void Depositar(decimal monto) {
        if (monto > 0)
            saldo += monto;
    }
}
```

## Composición vs Herencia

```csharp
// Composición — un Empleado "tiene una" Dirección
public class Empleado {
    public string Nombre { get; set; }
    public Direccion Direccion { get; set; }  // composición
}

public class Direccion {
    public string Calle { get; set; }
    public string Ciudad { get; set; }
}

// Herencia — un Empleado "es una" Persona
public class Empleado : Persona {
    // Hereda Nombre, Edad, etc.
}
```

## Método ToString()

Personalizar cómo se representa el objeto como texto.

```csharp
public class Persona {
    public string Nombre { get; set; }
    public int Edad { get; set; }
    
    public override string ToString() {
        return $"{Nombre} ({Edad} años)";
    }
}

var p = new Persona { Nombre = "Ana", Edad = 25 };
Console.WriteLine(p);  // "Ana (25 años)"
```

## Igualdad: Equals() y GetHashCode()

```csharp
public class Persona {
    public string Nombre { get; set; }
    
    public override bool Equals(object? obj) {
        if (obj is not Persona otra) return false;
        return this.Nombre == otra.Nombre;
    }
    
    public override int GetHashCode() {
        return Nombre.GetHashCode();
    }
}

var p1 = new Persona { Nombre = "Ana" };
var p2 = new Persona { Nombre = "Ana" };

Console.WriteLine(p1 == p2);      // false — son referencias distintas
Console.WriteLine(p1.Equals(p2)); // true — contenido igual
```

## Records — alternativa moderna

```csharp
// Record — inmutable y con Equals/GetHashCode automáticos
public record Persona(string Nombre, int Edad);

var p1 = new Persona("Ana", 25);
var p2 = new Persona("Ana", 25);

Console.WriteLine(p1 == p2);  // true — igualdad de valor
Console.WriteLine(p1.ToString());  // "Persona { Nombre = Ana, Edad = 25 }"

// Non-destructive update
var p3 = p1 with { Edad = 26 };
Console.WriteLine(p3);  // "Persona { Nombre = Ana, Edad = 26 }"
```

## Destructor y IDisposable

Para liberar recursos (archivos, conexiones, etc.).

```csharp
public class Recurso : IDisposable {
    private bool disposed = false;
    
    public void Dispose() {
        Dispose(true);
        GC.SuppressFinalize(this);
    }
    
    protected virtual void Dispose(bool disposing) {
        if (!disposed) {
            if (disposing) {
                // Liberar recursos gestionados
            }
            // Liberar recursos no gestionados
            disposed = true;
        }
    }
    
    ~Recurso() {  // Destructor
        Dispose(false);
    }
}

// Uso
using var recurso = new Recurso();
// Dispose() se llama automáticamente al salir del bloque
```
