# Interfaces y Contratos en C#

Definiendo comportamiento requerido, no implementación

---

## Qué es una interfaz

Una **interfaz** define un **contrato** — qué métodos y propiedades debe tener una clase.

```csharp
public interface IVehiculo {
    string Marca { get; }
    void Arrancar();
    void Detener();
}

public class Auto : IVehiculo {
    public string Marca { get; set; }
    
    public void Arrancar() {
        Console.WriteLine("Auto arrancando...");
    }
    
    public void Detener() {
        Console.WriteLine("Auto detenido");
    }
}
```

## Miembros de interfaz

```csharp
public interface IOperacion {
    // Propiedad (get/set)
    int Resultado { get; set; }
    
    // Propiedad solo lectura
    DateTime FechaCreacion { get; }
    
    // Método
    void Ejecutar();
    
    // Método que retorna valor
    bool Validar();
    
    // Método con parámetros
    void ProcesarDatos(int[] datos);
}
```

## Una clase puede implementar múltiples interfaces

```csharp
public interface IComparable {
    int CompareTo(object? obj);
}

public interface ICloneable {
    object Clone();
}

public class Producto : IComparable, ICloneable {
    public string Nombre { get; set; }
    public decimal Precio { get; set; }
    
    public int CompareTo(object? obj) {
        if (obj is not Producto otro) return 1;
        return this.Precio.CompareTo(otro.Precio);
    }
    
    public object Clone() {
        return new Producto { Nombre = this.Nombre, Precio = this.Precio };
    }
}
```

## Herencia de interfaces

Una interfaz puede heredar de otra:

```csharp
public interface IPersona {
    string Nombre { get; }
}

public interface IEmpleado : IPersona {
    decimal Salario { get; }
    void TrabajarEn(string proyecto);
}

// Implementar IEmpleado requiere implementar ambas interfaces
public class Empleado : IEmpleado {
    public string Nombre { get; set; }
    public decimal Salario { get; set; }
    
    public void TrabajarEn(string proyecto) {
        Console.WriteLine($"{Nombre} trabaja en {proyecto}");
    }
}
```

## Métodos por defecto en interfaces (C# 8+)

```csharp
public interface ILogger {
    void Log(string mensaje);
    
    // Implementación por defecto
    void LogError(string error) {
        Log($"[ERROR] {error}");
    }
    
    void LogInfo(string info) {
        Log($"[INFO] {info}");
    }
}

public class ConsoleLogger : ILogger {
    public void Log(string mensaje) {
        Console.WriteLine(mensaje);
    }
    // LogError y LogInfo ya tienen implementación
}
```

## Interfaces genéricas

```csharp
public interface IRepositorio<T> {
    T? ObtenerPorId(int id);
    List<T> ObtenerTodos();
    void Agregar(T elemento);
    void Eliminar(int id);
}

public class RepositorioProducto : IRepositorio<Producto> {
    private List<Producto> productos = [];
    
    public Producto? ObtenerPorId(int id) {
        return productos.FirstOrDefault(p => p.Id == id);
    }
    
    public List<Producto> ObtenerTodos() {
        return productos;
    }
    
    public void Agregar(Producto elemento) {
        productos.Add(elemento);
    }
    
    public void Eliminar(int id) {
        productos.RemoveAll(p => p.Id == id);
    }
}
```

## Interfaces como tipos de parámetro

```csharp
// Aceptar cualquier clase que implemente la interfaz
public static void Procesar(IVehiculo vehiculo) {
    vehiculo.Arrancar();
    Console.WriteLine("Viajando...");
    vehiculo.Detener();
}

var auto = new Auto();
var moto = new Moto();

Procesar(auto);   // ✓ Auto implementa IVehiculo
Procesar(moto);   // ✓ Moto implementa IVehiculo
```

## Casting a interfaz

```csharp
object obj = new Auto { Marca = "Toyota" };

// Verificar e intentar casting
if (obj is IVehiculo vehiculo) {
    vehiculo.Arrancar();
}

// Con as — devuelve null si no es compatible
IVehiculo? v = obj as IVehiculo;
if (v != null) {
    v.Arrancar();
}
```

## Interfaces comunes de .NET

**IEquatable<T>** — Definir igualdad:
```csharp
public class Persona : IEquatable<Persona> {
    public string Nombre { get; set; }
    
    public bool Equals(Persona? other) {
        return other is not null && this.Nombre == other.Nombre;
    }
}
```

**IComparable<T>** — Ordenamiento:
```csharp
public class Producto : IComparable<Producto> {
    public decimal Precio { get; set; }
    
    public int CompareTo(Producto? otro) {
        if (otro is null) return 1;
        return this.Precio.CompareTo(otro.Precio);
    }
}
```

**IEnumerable<T>** — Iteración:
```csharp
public class Coleccion : IEnumerable<int> {
    private int[] datos = [1, 2, 3, 4, 5];
    
    public IEnumerator<int> GetEnumerator() {
        foreach (var item in datos)
            yield return item;
    }
    
    IEnumerator IEnumerable.GetEnumerator() => GetEnumerator();
}
```

## Segregación de interfaces

Una clase puede implementar múltiples interfaces pequeñas:

```csharp
// Mejor: interfaces específicas
public interface ILeer {
    string Leer();
}

public interface IEscribir {
    void Escribir(string datos);
}

public class Archivo : ILeer, IEscribir {
    public string Leer() => "contenido del archivo";
    public void Escribir(string datos) => Console.WriteLine($"Escribiendo: {datos}");
}

// En lugar de una interfaz gigante
public interface IArchivo {  // ✗ demasiado grande
    string Leer();
    void Escribir(string datos);
    void Eliminar();
    void Renombrar(string nuevoNombre);
    // ... 10 métodos más
}
```

## Diferencias: Clase abstracta vs Interfaz

| Aspecto              | Clase abstracta       | Interfaz          |
|----------------------|----------------------|-------------------|
| Herencia múltiple    | Solo una             | Múltiples         |
| Implementación       | Puede tener          | Solo parámetros   |
| Campos               | Sí                   | No (C# 7)         |
| Constructor          | Sí                   | No                |
| Modificadores access | public/private/etc   | public (implícito)|
| Uso                  | Jerarquía            | Contrato          |

## Interfaz explícita

Implementar un método dos formas distintas para dos interfaces:

```csharp
public interface IA {
    void Metodo();
}

public interface IB {
    void Metodo();
}

public class Clase : IA, IB {
    // Implementación explícita
    void IA.Metodo() {
        Console.WriteLine("IA.Metodo");
    }
    
    void IB.Metodo() {
        Console.WriteLine("IB.Metodo");
    }
}

var obj = new Clase();
// obj.Metodo();  // ✗ Error — ambiguo
((IA)obj).Metodo();   // ✓ "IA.Metodo"
((IB)obj).Metodo();   // ✓ "IB.Metodo"
```
