# Strings en C#

Uno de los tipos más usados y con comportamientos especiales

---

## Qué es `string`

Representa texto. Es un tipo de **referencia** pero con comportamiento especial (inmutable).

```cs
string nombre = "Ada";
string mensaje = "Hola mundo";
```

---

## Formas de construir un string

**Concatenación:**
```cs
string saludo = "Hola, " + nombre;
```

**Interpolación** (preferida):
```cs
string mensaje = $"{nombre} tiene {edad} años";
string resultado = $"El doble de {edad} es {edad * 2}";
```

**Verbatim strings** (rutas, multilínea):
```cs
string ruta = @"C:\Users\Ada\Documentos";

string texto = @"Línea 1
Línea 2
Línea 3";

// Combinado
string carpeta = "Documentos";
string ruta2 = $@"C:\Users\{carpeta}";
```

---

## Raw string literals para JSON

```cs
// Literales crudos — comillas sin escapar
string json = """
    {
        "nombre": "Ada",
        "edad": 20,
        "activo": true
    }
    """;

// Con interpolación ($$)
string nombre = "Ada";
string json2 = $$"""
    {
        "nombre": "{{nombre}}",
        "edad": {{20}}
    }
    """;
```

---

## Inmutabilidad

Los strings son **inmutables**. Una vez creados, su contenido no cambia.

```cs
string nombre = "Ana";
nombre = nombre + " María";  // crea un nuevo string
```

No se edita el original; se genera uno nuevo. El original existe en memoria hasta recolección de basura.

**`StringBuilder` para construcción intensiva:**
```cs
var sb = new StringBuilder();
for (int i = 0; i < 1000; i++)
    sb.Append($"Línea {i}\n");
string resultado = sb.ToString();
```

## Comparación de strings

**Igualdad:**
```cs
string a = "hola";
string b = "hola";
Console.WriteLine(a == b);  // true — compara contenido
```

**Orden (CompareTo):**
```cs
string a = "banana";
string b = "manzana";
int resultado = a.CompareTo(b);  // negativo, cero, o positivo
```

**Sin importar mayúsculas:**
```cs
string frase = "El Cielo es Azul";
bool igual = frase.Equals("el cielo es azul", StringComparison.OrdinalIgnoreCase);
bool contiene = frase.Contains("cielo", StringComparison.OrdinalIgnoreCase);
```

## Métodos comunes

```cs
string texto = "Hola Mundo";

texto.Length;              // cantidad de caracteres
texto.ToUpper();           // "HOLA MUNDO"
texto.ToLower();           // "hola mundo"

"  hola  ".Trim();         // "hola"
"  hola  ".TrimStart();    // "hola  "
"  hola  ".TrimEnd();      // "  hola"

frase.Contains("azul");    // true/false
frase.StartsWith("el");    // true/false
frase.EndsWith("azul");    // true/false
frase.IndexOf("mundo");    // posición o -1

texto.Replace("Mundo", "C#");  // reemplazo

"Ana,Luis,Pedro".Split(',');   // array

string[] nombres = { "Ana", "Luis", "Pedro" };
string.Join(", ", nombres);    // "Ana, Luis, Pedro"

texto.Substring(0, 4);     // "Hola"
texto[0..4];               // "Hola" (rango)
```

## Null vs vacío

```cs
string? nulo = null;
string vacio = "";
string blanco = "   ";

// Ambos null y vacío
string.IsNullOrEmpty(nulo);   // true
string.IsNullOrEmpty(vacio);  // true

// Detectar también espacios
string.IsNullOrWhiteSpace(blanco);  // true
```

## Unicode, emojis y `Rune`

En C#, `string` usa **UTF-16** internamente. Los emojis modernos ocupan **2 caracteres**:

```cs
string emoji = "😀";
Console.WriteLine(emoji.Length);  // 2 (no 1)
```

**Recorrer correctamente con `Rune`:**
```cs
string texto = "A😀B";
foreach (Rune r in texto.EnumerateRunes())
    Console.WriteLine(r);  // [A] [😀] [B]
```

**Medir longitud visible:**
```cs
var info = new StringInfo("Hola 😀");
Console.WriteLine(info.LengthInTextElements);  // 6
```

## Resumen de métodos útiles

| Método                            | Qué hace                                  |
|-----------------------------------|-------------------------------------------|
| `Length`                          | Cantidad de unidades `char`               |
| `ToUpper()` / `ToLower()`         | Convierte a mayúsculas/minúsculas         |
| `Trim()` / `TrimStart()` / `TrimEnd()` | Elimina espacios en extremos        |
| `Contains(sub)`                   | Verifica si contiene un fragmento         |
| `StartsWith()` / `EndsWith()`     | Verifica inicio o final                   |
| `IndexOf(sub)`                    | Posición primera aparición                |
| `Replace(viejo, nuevo)`           | Reemplaza todas las apariciones           |
| `Split(sep)`                      | Divide en array                           |
| `string.Join(sep, array)`         | Une array con separador                   |
| `Substring(inicio, largo)`        | Extrae una parte                          |
| `IsNullOrEmpty(s)`                | `true` si null o `""`                     |
| `IsNullOrWhiteSpace(s)`           | `true` si null, `""` o solo espacios      |
