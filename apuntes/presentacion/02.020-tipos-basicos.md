# Tipos de Datos y Operadores en C#

Los bloques fundamentales de cualquier programa

---

## Tipos primitivos

| Tipo     | Representa            | Ejemplo       |
|----------|----------------------|---------------|
| `int`    | Número entero        | 20, -5        |
| `double` | Decimal (doble)      | 3.14, 1.75    |
| `bool`   | Valor lógico         | true, false   |
| `char`   | Un carácter          | 'A', 'ñ'      |

---

## Literales y sufijos

La forma del literal indica el tipo:

```cs
int edad = 20;               // sin sufijo → int
long distancia = 5_000_000_000L;  // L → long
double precio = 19.99;       // punto decimal → double
float temperatura = 25.5f;   // f → float
decimal saldo = 1500.25m;    // m → decimal
```

Hexadecimal y binario:
```cs
int color = 0xFF_EC_DE;      // hex
int flags = 0b1111_0000;     // binario
```

---

## Tipos enteros

| Tipo    | Rango                      |
|---------|----------------------------|
| `byte`  | 0 a 255                    |
| `sbyte` | -128 a 127                 |
| `short` | -32.768 a 32.767           |
| `int`   | ±2.1 mil millones          |
| `long`  | ±9.2 × 10¹⁸                |

En la práctica: casi siempre `int`. Cambiar solo si hay razón.

```cs
Console.WriteLine(int.MaxValue);   // 2.147.483.647
Console.WriteLine(int.MinValue);   // -2.147.483.648
```

---

## Punto flotante y decimal

| Tipo      | Precisión     | Sufijo | Uso                      |
|-----------|---------------|--------|--------------------------|
| `float`   | ~7 dígitos    | f      | Gráficos, juegos         |
| `double`  | ~15 dígitos   | d      | Uso general              |
| `decimal` | 28-29 dígitos | m      | Dinero, finanzas         |

⚠️ **Problema de punto flotante:**
```cs
double resultado = 0.1 + 0.2;
Console.WriteLine(resultado);  // 0.30000000000000004
```

Usar `decimal` para dinero.

## Booleanos

El tipo `bool` solo tiene dos valores: `true` y `false`.

```cs
bool activo = true;
bool esMayor = 10 > 5;     // true
bool esIgual = 3 == 3;     // true
```

**Operadores lógicos:**
```cs
bool y = c && d;   // AND — ambas verdaderas
bool o = c || d;   // OR — al menos una verdadera
bool no = !c;      // NOT — negación
```

El cortocircuito evita errores:
```cs
if (x != 0 && (10 / x) > 1)  // (10/x) no se evalúa si x==0
```

## Caracteres (`char`) y Unicode

Representa exactamente un carácter:

```cs
char letra = 'A';
char acento = 'ñ';
char euro = '€';
```

**Diferencia con string:**
```cs
char c = 'A';     // un carácter
string s = "A";   // una cadena
```

**Secuencias de escape:**
```cs
char tab = '\t';
char linea = '\n';
char barra = '\\';
```

## Operadores aritméticos

```cs
int a = 10;
int b = 3;

int suma = a + b;              // 13
int resta = a - b;             // 7
int multi = a * b;             // 30
int division = a / b;          // 3 (entero)
int modulo = a % b;            // 1
```

⚠️ **División entera descarta decimales:**
```cs
double resultado = 10.0 / 3;   // 3.333...
double resultado2 = (double)10 / 3;  // casting
```

---

## Incremento y decremento

```cs
int x = 5;

x++;   // postincremento: usa x (5) luego incrementa
++x;   // preincremento: incrementa primero
x--;   // postdecremento
--x;   // predecremento

// Diferencia en expresiones:
int a = 5;
int b = a++;   // b = 5, a = 6
int c = ++a;   // c = 7, a = 7
```

---

## Operadores de comparación

```cs
bool igual = a == b;
bool distinto = a != b;
bool mayor = a > b;
bool menor = a < b;
bool mayorIgual = a >= b;
bool menorIgual = a <= b;
```

**Importante para tipos por referencia:**
- `==` sobre tipos por valor → compara contenido
- `==` sobre tipos por referencia → compara si apuntan al mismo objeto

---

## Operadores bit a bit

```cs
int a = 0b1100;    // 12
int b = 0b1010;    // 10

int and = a & b;   // 0b1000 = 8
int or = a | b;    // 0b1110 = 14
int xor = a ^ b;   // 0b0110 = 6
int not = ~a;      // invierte todos los bits

int left = a << 1;   // 0b11000 = 24 (× 2)
int right = a >> 1;  // 0b0110 = 6 (÷ 2)
```

## Conversión de tipos

**Implícita** (segura, automática):
```cs
int i = 10;
long l = i;        // int → long: seguro
double d = i;      // int → double: seguro
```

**Explícita** (casting):
```cs
double precio = 19.99;
int entero = (int)precio;  // 19 (descarta decimales)

long grande = 5_000_000_000L;
int normal = (int)grande;  // puede desbordarse
```

**Métodos auxiliares:**
```cs
int.Parse("42");           // lanza excepción si falla
int.TryParse("42", out int n);  // devuelve bool
Convert.ToInt32(3.7);      // redondea
```

---

## Tipo por valor vs por referencia

| Aspecto          | Por valor  | Por referencia |
|------------------|-----------|----------------|
| Cómo se copia    | El dato   | La referencia  |
| Asignación       | Copia     | Apunta al objeto |
| Cambios visibles | No        | Sí            |

`string` es referencia pero parece valor (inmutable).
