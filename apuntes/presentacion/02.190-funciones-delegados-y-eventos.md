# Funciones, Delegados y Eventos en C#

Las funciones como valores

---

## Declaración básica de funciones

```csharp
// Forma completa
static int Sumar(int a, int b) {
    return a + b;
}

// Expresión de cuerpo (una línea)
static int Sumar(int a, int b) => a + b;

// Sin parámetros ni retorno
static void Limpiar() => Console.Clear();

// Sin parámetros, con retorno
static string Saludar() => "Hola";
```

---

## Parámetros especiales

**`ref`** — Pasar por referencia verdadera:
```csharp
static void Incrementar(ref int valor) => valor += 1;

int contador = 10;
Incrementar(ref contador);
Console.WriteLine(contador);  // 11
```

**`out`** — Retorno por parámetro:
```csharp
static bool TryParseFecha(string texto, out DateOnly fecha) {
    return DateOnly.TryParse(texto, out fecha);
}

if (TryParseFecha("2025-06-15", out DateOnly fecha))
    Console.WriteLine(fecha);
```

**`params`** — Número variable de argumentos:
```csharp
static int Sumar(params int[] numeros) => numeros.Sum();

Console.WriteLine(Sumar(1, 2, 3, 4, 5));  // 15
```

## Funciones locales (anidadas)

```csharp
static List<int> FiltrarYOrdenar(IEnumerable<int> fuente, int minimo) {
    // Función local — visible solo aquí
    bool EsValido(int n) => n >= minimo && n % 2 == 0;
    
    return fuente.Where(EsValido).OrderBy(n => n).ToList();
}
```

**Con captura de variables:**
```csharp
static void Procesar(List<Orden> ordenes, decimal tasaIva) {
    // tasaIva es capturada
    decimal CalcularTotal(Orden o) => o.Subtotal * (1 + tasaIva);
    
    foreach (var orden in ordenes)
        Console.WriteLine(CalcularTotal(orden));
}
```

## Recursión

Una función que se llama a sí misma:

```csharp
// Factorial
static long Factorial(int n) {
    if (n <= 1) return 1;        // caso base
    return n * Factorial(n - 1);  // llamada recursiva
}

// Fibonacci
static long Fibonacci(int n) => n switch {
    < 0 => throw new ArgumentException(),
    0   => 0,      // caso base
    1   => 1,      // caso base
    _   => Fibonacci(n - 1) + Fibonacci(n - 2)
};
```

## Delegados

Un **delegado** es una variable que guarda una función.

```csharp
// Declarar el tipo de delegado
delegate int Operacion(int a, int b);

// Métodos compatibles
static int Sumar(int a, int b) => a + b;
static int Restar(int a, int b) => a - b;

// Guardar una función en una variable
Operacion op = Sumar;
Console.WriteLine(op(3, 4));  // 7

op = Restar;
Console.WriteLine(op(10, 3)); // 7
```

---

## Multicast — múltiples métodos en un delegado

```csharp
delegate void Notificacion(string msg);

static void Email(string msg) => Console.WriteLine($"[Email] {msg}");
static void SMS(string msg) => Console.WriteLine($"[SMS] {msg}");
static void Log(string msg) => Console.WriteLine($"[Log] {msg}");

Notificacion notificar = Email;
notificar += SMS;   // agregar
notificar += Log;

notificar("Pedido confirmado");
// [Email] Pedido confirmado
// [SMS] Pedido confirmado
// [Log] Pedido confirmado

notificar -= SMS;   // quitar
```

## `Func<>`, `Action<>`, `Predicate<>`

Delegados genéricos del sistema:

```csharp
// Func — función que retorna valor
// Último parámetro = tipo de retorno
Func<int, int, int> sumar = (a, b) => a + b;
Func<string, int> largo = s => s.Length;

// Action — función void (sin retorno)
Action<string> imprimir = s => Console.WriteLine(s);
Action limpiar = () => Console.Clear();

// Predicate — función que retorna bool
Predicate<int> esPar = n => n % 2 == 0;
Predicate<string> estaVacio = string.IsNullOrWhiteSpace;
```

## Expresiones lambda

Funciones anónimas de forma compacta:

```csharp
// Una expresión
Func<int, int> cuadrado = x => x * x;

// Múltiples instrucciones
Func<int, int, int> maximo = (a, b) => {
    if (a > b) return a;
    return b;
};

// Sin parámetros
Action saludar = () => Console.WriteLine("Hola");

// Captura de variables
int base_ = 10;
Func<int, int> sumarBase = n => n + base_;
```

## Funciones de orden superior

**Recibir funciones como parámetro:**
```csharp
static List<TResult> Transformar<T, TResult>(
    List<T> lista, 
    Func<T, TResult> transformacion)
    => lista.Select(transformacion).ToList();

var numeros = new List<int> { 1, 2, 3, 4, 5 };
var dobles = Transformar(numeros, n => n * 2);  // [2, 4, 6, 8, 10]
```

**Devolver funciones:**
```csharp
static Func<int, int> CrearMultiplicador(int factor) 
    => n => n * factor;

var doble = CrearMultiplicador(2);
var triple = CrearMultiplicador(3);

Console.WriteLine(doble(5));   // 10
Console.WriteLine(triple(5));  // 15
```

## Composición de funciones

```csharp
// Componer dos funciones
static Func<T, TResult> Componer<T, TMedio, TResult>(
    Func<T, TMedio> f,
    Func<TMedio, TResult> g)
    => entrada => g(f(entrada));

Func<string, string> limpiar = s => s.Trim().ToLower();
Func<string, string[]> dividir = s => s.Split(' ');
Func<string[], string> unir = arr => string.Join("-", arr);

var slug = Componer(Componer(limpiar, dividir), unir);
Console.WriteLine(slug("  Hola Mundo  "));  // hola-mundo
```

## Eventos

Un **evento** permite que un objeto notifique a otros cuando algo ocurre.

```csharp
public class Temporizador {
    public event EventHandler? Tick;
    
    protected virtual void OnTick() {
        Tick?.Invoke(this, EventArgs.Empty);
    }
    
    public async Task Iniciar(int segundos) {
        for (int i = 1; i <= segundos; i++) {
            await Task.Delay(1000);
            OnTick();
        }
    }
}

// Suscribirse
var timer = new Temporizador();
timer.Tick += (sender, e) => Console.WriteLine("Tick!");
await timer.Iniciar(3);
```

## EventHandler<TEventArgs> — patrón estándar

```csharp
// 1. Datos del evento
public class PedidoCreadoEventArgs(string cliente, decimal total) 
    : EventArgs {
    public string Cliente => cliente;
    public decimal Total => total;
}

// 2. Publicador
public class SistemaPedidos {
    public event EventHandler<PedidoCreadoEventArgs>? PedidoCreado;
    
    public void CrearPedido(string cliente, decimal total) {
        var args = new PedidoCreadoEventArgs(cliente, total);
        PedidoCreado?.Invoke(this, args);
    }
}

// 3. Suscriptor
var sistema = new SistemaPedidos();
sistema.PedidoCreado += (sender, e) => 
    Console.WriteLine($"Nuevo pedido de {e.Cliente}: ${e.Total}");

sistema.CrearPedido("Ana García", 15_000);
```

## Diferencias: Delegado vs Evento

| Aspecto          | Delegado                | Evento                 |
|-----------------|------------------------|------------------------|
| Quién invoca    | Cualquiera              | Solo la clase que lo declara |
| Asignación `=`  | Cualquiera              | Solo la clase que lo declara |
| Suscripción `+=`| Cualquiera              | Cualquiera             |
| Uso típico      | Callback directo        | Publicador-suscriptor  |

```csharp
// Con delegado — peligroso
sistema.Alguno = null;  // borra todas las suscripciones

// Con evento — solo += y -=
sistema.PedidoCreado += handler;
sistema.PedidoCreado -= handler;
// sistema.PedidoCreado = null;  // ✗ Error
```

## Callbacks — ejemplo práctico

```csharp
static async Task DescargarArchivo(
    string url,
    Action<int> onProgreso,
    Action<string> onExito,
    Action<Exception> onError) {
    try {
        onProgreso(10);
        var contenido = await new HttpClient().GetByteArrayAsync(url);
        onProgreso(80);
        
        var ruta = Path.GetTempFileName();
        await File.WriteAllBytesAsync(ruta, contenido);
        onProgreso(100);
        onExito(ruta);
    } catch (Exception ex) {
        onError(ex);
    }
}

// Uso
await DescargarArchivo(
    "https://ejemplo.com/datos.csv",
    pct => Console.WriteLine($"Descargando... {pct}%"),
    ruta => Console.WriteLine($"Guardado en: {ruta}"),
    ex => Console.WriteLine($"Error: {ex.Message}")
);
```
